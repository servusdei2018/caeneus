# Security Policy

## Supported Versions

The latest version of Caeneus is maintained with current security patches as they arise.

| Version | Supported          |
| ------- | ------------------ |
| 0.0.x   | :white_check_mark: |

## Reporting a Vulnerability

To report a security vulnerability, please open a new [Issue](https://github.com/servusdei2018/caeneus/issues) and we will look at it as quickly as possible.
